## Overview
This is my first Solo Project from Scrimba! The task was to get familiar with the Math Random method to create a two passwords simultaniously. I wanted to include differenct characters to the password so I added few options. I was able to create the function with issues but I really struggle associating the variables representing the character set and the checkboxes options. I based my solution on an other project so I feel like I really couldn't figuered out by myself. I added a copy password buttons and a toast notification to let the user know when the password was coppied. Happy Coding!
